# TOKIMEKI Bluesky

An unofficial Bluesky web client.

Supporting multiple PDS servers.

## Features

- [x] Show timeline
- [x] Show user profiles
- [x] Create / Delete post
- [x] Reply
- [x] Follow / Unfollow
- [x] Notifications
- [x] Vote (Like)
- [x] Repost
- [x] Upload images
- [x] Quote post
- [x] Mention
- [ ] Create post with entities formatting
- [x] Update profile
- [ ] Create account
